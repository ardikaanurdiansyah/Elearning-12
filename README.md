# Elearning-12  
## Latihan CRUD Flutter (Local & Cloud)

Aplikasi mobile CRUD menggunakan Flutter dengan dua metode penyimpanan:
- Local Storage (SharedPreferences)
- Cloud Storage (Firebase Firestore)

## Fitur
- Create, Read, Update, Delete (CRUD)
- Pilih penyimpanan Local atau Cloud
- Integrasi Firebase Firestore
